# Weather_Forecast_Website_React
A sleek, real-time weather application built with React.js and OpenWeatherMap API. This project demonstrates how to handle API calls, manage state, and implement responsive design with CSS.

Features

Live Weather Data: Fetches current temperature and weather conditions based on your location.

5-Day Forecast: Displays a clear, daily forecast for the week ahead.

City Search: Easily search for any city globally to get instant weather updates.

Glassmorphism UI: Features a modern, translucent design with smooth CSS gradients.

The "easy way" to run your project locally is essentially the standard procedure for all React apps.
Here is the quick 3-step routine.

1. The "Daily" Startup (Fastest way)
If you already have the code on your computer and just want to see it running again:

Open your Command Prompt (CMD) or the VS Code Terminal.

2.Type this to go to your project folder:
cd Desktop/FirstApp/weather-app

3.Type this to launch the app:
npm start

Your browser will open automatically to localhost:3000.
